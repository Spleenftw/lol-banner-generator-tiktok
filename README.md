# Instructions 2 :
```
git clone  https://github.com/Spleenftw/lol-banner-generator-tiktok
cd lol-banner-generator-tiktok
docker build -t lol-react-app .
docker run -d -p 3002:3002 --name lol-app lol-react-app
```
