# lol-banner-generator-tiktok
Fully vibecoded shit(tool) via GeminiAI that worked more than once.
<br>
Use case : generate a banner for my league of legends tiktoks using riot games API.

# Instructions 1 
Generate your RIOT GAMES API KEY at https://developer.riotgames.com/ and add it in server.js:14

# Instructions 2 :
```
git clone https://github.com/Spleenftw/lol-banner-generator-tiktok.git
cd lol-banner-generator-tiktok
docker build -t lol-react-app .
docker run -d -p 3002:3002 --name lol-app lol-react-app
```

# Access :
Visit http://ip:3002, enter your league of legends tag and generate your own banner for tiktoks.
